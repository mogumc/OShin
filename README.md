### 🔧 兼容OpenList生态  
本项目**基于 [openlist](https://github.com/OpenListTeam/OpenList) 项目**并继承其 [AGPLv3.0](https://www.gnu.org/licenses/agpl-3.0.html)许可证。  
不同于官方或其他实现通过 API 交互的方式提供本地化版本，本项目将使用 [Wails v2](https://wails.io) 替换原有的 [Gin](https://github.com/gin-gonic/gin) 框架，**在本地直接运行类似 OpenList 的服务**。

这将带来以下优势：
- ✅ 全功能的本地化体验
- ✅ 更灵活的本地能力
- ✅ 贴近原生桌面应用的性能表现
- ✅ 支持接入AI进行文件管理
